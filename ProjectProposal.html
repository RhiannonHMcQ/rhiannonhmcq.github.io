Rhiannon's Final Project Proposal!


Description

Project Concept/Motivation
I enjoy listening to the radio (either for the news or for music) and would love a more 'old-school' way to do this, rather than using my phone or laptop. For this project, I will create a radio that, instead of using radio waves, uses an internet connection to pull up different online streams of radio stations.

Anticipated Bill of Materials

  • Speaker/s for sound
  • Assorted resistors for current limiting
  • Wifi enabled microcontroller- seeed xiao ESP3253
  • Potentiometer for changing stations
  • Physical containment for a radio shell
  • LCD screen to display current station, and other information
  • Battery and connection to power radio
  • Transistor to limit battery power

Timeline and Contingency Plans

  • Major Milestones
    1. Connect microcontroller and knob to change stations
    2. Add LCD screen for display
    3. Add speaker for sound
    4. Create physical container
  • If the microcontroller doesn't work as intended, I will consider creating a radio using an Arduino Uno that needs connection to a computer instead of using bluetooth.
